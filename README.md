### 文件介绍
Builder      : 建造者模式

config       : 配置文件, 设置图标族

FJEVisitor   : 抽象访问者和具体访问者

Iterator     : 迭代器模式

JSONComponent: 组合模式

main         : 客户端

### 执行方式：
```bash
python main.py -f test.json -s [tree/rect] -i [default/fruit/animal/sports]
```
e.g.
```bash
python main.py -f test.json -s tree -i fruit
```
